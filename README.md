#### How to use

Use at least v6.0.0 of NodeJS.

```bash

```

Bonus Sample call:

#### Technologies

- Hosting: zeit/now
- DB: mLab
- API: Hapi
- Canvas: EaselJS
- Bundling: webpack
- Flow control: Bluebird, tj/co
- Code style: feross/standard
